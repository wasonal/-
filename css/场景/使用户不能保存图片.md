### 使用户不能保存图片
```
<img src="/i/eg_tulip.jpg"  alt="郁金香" height="100px"/>
<img src="/i/eg_tulip.jpg"  alt="郁金香" height="100px"/>
```
### 解决方法
1. user-select  

属性值 | 描述
:-- | :--
auto | 
none | 元素及其子元素不可选中
text | 文本可选中
all | 当双击子元素或者上下文时，那么包含该子元素的最顶层元素也会被选中
contain | 可以选择文本，但选择范围受元素边界的约束
设置user-select:none;为了兼容不同内核，需要加上前缀
```
-webkit-user-select: none;
-moz-user-select: none;
-ms-user-select: none;
user-select: none;
```
